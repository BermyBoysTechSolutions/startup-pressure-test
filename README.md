## What is this?

A framework for founders, operators, and operators-who-are-about-to-become-founders to pressure-test startup ideas **before** wasting months building the wrong thing.

## Usage

Drop this into any AI agent that supports custom skills. The agent reads the `SKILL.md` and `references/playbooks.md`, then follows the framework to give you a brutal, structured diagnosis.

## Modes

| Mode | When to use |
|------|-------------|
| `pressure-test` | Find fatal flaws and get a verdict |
| `problem-validation` | Test whether the pain is real and urgent |
| `competition-map` | Map current behavior, direct/indirect competitors |
| `first-10-customers` | Build a manual plan to find your first users |
| `mvp-plan` | Define the smallest MVP to test the core assumption in 2 weeks |
| `full` | Run all modes compactly |

## Quick start

Tell your agent:

```
Use the startup-pressure-test skill. Mode: full.
Idea: [your idea]
Target customer: [who you're building for]
What you want them to do/pay for: [the action/payment]
```
